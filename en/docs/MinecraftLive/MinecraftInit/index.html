Aqui le muestro la sintaxis de minecraft :

chat‚Äã

you can configure the commands by the comments they send The chat syntax is as follows:

/say este es un comando del chat
/say uniqueId Comment

gift‚Äã

you can configure the commands by the gifts you send The syntax of gift is as follows

example default
  default:
/say este es un comando de regalos que envia uniqueId
/say uniqueId dio giftName xrepeatCount
example de Rose
  Rose:
/say este es comando de giftName
/say uniqueId regalo giftName xrepeatCount

follow‚Äã

you can configure the commands by which you are followed The syntax of follow is as follows:

/tellraw @a {\"text\":\"uniqueId  te sige \", \"color\":\"gold\"}
/say este es un comando de follow
/say uniqueId  te sige

share‚Äã

puede configurar los comandos por los que te comparten The syntax of share is as follows:

/tellraw @a {\"text\":\"uniqueId  compartio \", \"color\":\"gold\"}
/say el comando anterior muestra cuando comparten

welcome‚Äã

puede configurar los comandos por los que te ingresan al Live The syntax of WELCOME is as follows:

  welcome:
/say join uniqueId

cada vez que alguien ingresa el compando anterior nos da una manzana dorada

envelope‚Äã

puede configurar los comandos si envian un cofre The syntax of envelope is as follows:

envelope:
