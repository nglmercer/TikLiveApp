Crea un bot para interactuar en un servidor

Requerimientos anteriores​

Puede configurar el bot para definir su:

  • nombre del bot
  • Ip del server : puerto
  • Su nombre del jugador minecraft
  • comando inicial
  • Eventos como:
  1. chat
  2. gift
  3. follow
  4. share
