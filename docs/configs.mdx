## Configuracion extras
Utulidades
- Importar Configuracion:
Importa las Configuraciones de un json que se guardan en local storage como:
1. Audios
2. datos como usuario, palabras filtradas, nombres y demas.
3. algunas Configuraciones anteriores como tema, estados

- Exportar Configuracion: 
Exporta las Configuracion en un settings.json
 para que pueda importarlo con *Importar Configuracion* con todos los datos mencionados.
